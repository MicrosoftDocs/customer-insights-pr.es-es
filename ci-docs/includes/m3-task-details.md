Para comprobar el estado del proceso, seleccione **En actualización**. El panel **Detalles de la tarea** se muestra. Debajo de cada tarea, puede seleccionar **Ver detalles** para obtener más información sobre el progreso.

Los resultados de una ejecución exitosa se muestran en la página **Unificar** que muestra el número de perfiles de clientes unificados.
